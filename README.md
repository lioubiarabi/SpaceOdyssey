## 📄 Pages

| Page | Description |
|---|---|
| 🏠 **Home** | Hero banner, intro to space exploration, image of the day, team section, features overview |
| 🪐 **Planets** | All solar system planets with images, descriptions, and key data (moons, temperature, day length) |
| 🚀 **Missions** | 10+ famous space missions (Apollo, Curiosity, Artemis, James Webb...) with agency, objective, and launch date |
| ℹ️ **About** | AstroTech Studio presentation, mission, and the imaginary team behind the project |
| 📬 **Contact** | Contact form (name, email, message) + social media links |
| ✅ **Success** | Confirmation page after form submission |

---

## ✨ Features

- 🌠 Space-themed design with consistent visual identity (colors, typography, layout)
- 📱 Fully responsive — tested on mobile, tablet, and desktop
- 🎨 Hover effects, CSS transitions, and smooth animations
- 🧭 Intuitive navigation across all pages
- 📝 Clean, semantic HTML with commented code
- 🌍 Hosted and deployed via GitHub Pages

## 🎯 Project Context

This project was built as part of the **[2023] Développeur Web et Web Mobile** curriculum at **Youcode**.

The brief: join the Front-End team at AstroTech Studio to complete a partial Figma mockup and fully integrate a 5-page astronomy website — with responsive design, visual consistency, and interactive effects.

**Duration:** 5 days (Oct 13–17, 2025)
